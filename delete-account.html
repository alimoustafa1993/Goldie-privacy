Delete Your Account

If you wish to delete your Goldie Steps account and all associated data, please send an email request to:

agccurrency@gmail.com

Please send the deletion request from the same Google account that you used to sign in to the app, so we can verify account ownership.

What data will be deleted

  • User profile information
  • Authentication identifiers (Google Sign-In data)
  • Application usage data associated with the account

Data retention

Some data may be retained for legal or security purposes for up to 30 days. After this period, all remaining data will be permanently deleted.

Guest Mode

If you are using the app in Guest Mode, no personal account data is stored, and no deletion request is required.
